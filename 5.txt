Создание ветки
git branch имя_ветки
Переключиться на ветку
git checkout имя_ветки
Создать ветку и переключиться на нее
git checkout -b имя_ветки
"смерживание" веток
git merge имя_ветки
Удалить ветку (если она полностью слита с какой-то другой веткой)
git branch -d имя_ветки
Удалить любую локальную ветку
git branch -D имя_ветки
Отправить ветку на сервер
git push origin имя_ветки
Удалить ветку с сервера
git push origin  - -delete имя_ветки
Стянуть информацию о ветках на сервере к нам на локальную машину
git fetch 
Создать в удаленном репозитории ветку, соответствующую локальной ветке, и обе связать
git push -u имя_ветки 
Вывести список локальных веток
git branch 
Показать все существующие ветки и в локальном, и удаленном репозитории
git branch -a
Показать удаленные ветки на сервере
git branch -r
