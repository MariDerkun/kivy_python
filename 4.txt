Добавить файлы в индекс
git add имя_файла
Добавить все файлы в индекс
git add *
Удалить файлы из индекса
git reset 
Удалить конкретный файл из индекса
git restore  - -staged имя_файла 
Сделать коммит
git commit -m "Комментарий к коммиту"
Отменить все коммиты после заданного, оставляя все изменения в рабочем каталоге 
git reset хэш_коммита
Сбросить всю историю вместе с состоянием рабочего каталога до указанного коммита 
git reset  - -hard хэш_коммита  
Переключиться на любой коммит в истории наших изменений
git checkout хэш_коммита
Отложить изменения, не добавляя их в коммит
git stash
Показать список всех отложенных изменений
git stash list
Забрать данные из stash и удалить их
git stash pop
Забрать данные из stash, но не удалять их
git stash apply 
Удалить все файлы, которые находятся в stash
git stash clear 
Взять коммит с одной ветки и повторить его на другой ветке
git cherry -pick хэш_коммита
