Отобразить список измененных, добавленных и удаленных файлов
git status
Посмотреть историю коммитов
git log
Ограничить количество отображаемых коммитов этим числом
git log -число
Отобразить разницу, внесенную в каждый коммит
git log -p
Показать краткую статистику по каждому коммиту
git log –stat
Отобразить каждый коммит в одну строку
git log  - -oneline
Найти коммиты определенного автора
git log  - -author="Кот Бантик"
Посмотреть отдельный коммит
git show хэш_коммита
Показать разницу между рабочей копией и индексом
git diff  
Показываетать разницу между индексом и последним коммитом
git diff  - -staged
